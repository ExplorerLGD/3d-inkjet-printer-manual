# Klipper Notes

## Klipper running as root in the image

In the provided image files, Klipper is run with root privileges. Do not update Klipper, Fluidd, or other printing-related programs unless you know what you are doing. Incorrect updates may break the printing services or cause configuration loss.

## Configuration alignment

Most 3D-printing related configurations in this repository follow the Klipper official documentation. For configuration details and best practices, see the Klipper documentation:

- Klipper documentation: https://www.klipper3d.org/
