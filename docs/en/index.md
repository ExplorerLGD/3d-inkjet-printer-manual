# Overview

This development kit is designed for DIY 3D printing enthusiasts. It provides a complete solution covering hardware, materials, and slicing software.

To make the system easier for users who already have Klipper experience, this design follows the Klipper 3D printer firmware architecture and only makes the necessary modifications, such as running Klipper with root permissions.

For users who have purchased the kit, we will open source the host controller hardware design, host controller source code, and mechanical structure design, and provide free full-color slicing software.
